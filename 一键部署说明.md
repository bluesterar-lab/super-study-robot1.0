# 🎯 一键部署！超简单！

## 💻 Windows 用户

### 只需要 3 步：

#### 第 1 步：双击运行
在文件夹中找到 `auto-deploy.bat` 文件，**双击运行**

#### 第 2 步：登录 Vercel
- 会自动打开浏览器
- 用 GitHub 或邮箱登录（免费）
- 登录后返回命令行

#### 第 3 步：等待完成
- 脚本会自动完成所有操作
- 等待 3-5 分钟
- 会显示你的网站链接

**完成！打开那个链接就能看到你的网站！** 🎉

---

## 🍎 Mac/Linux 用户

### 只需要 3 步：

#### 第 1 步：打开终端
在终端中进入项目文件夹

#### 第 2 步：运行脚本
```bash
./auto-deploy.sh
```

#### 第 3 步：登录 Vercel
- 会自动打开浏览器
- 用 GitHub 或邮箱登录（免费）
- 登录后返回终端

**完成！等待 3-5 分钟，会显示你的网站链接！** 🎉

## ❓ 遇到问题？

### 问题 1：脚本无法运行
**Windows**：右键点击脚本 → 以管理员身份运行

**Mac/Linux**：
```bash
chmod +x auto-deploy.sh
./auto-deploy.sh
```

### 问题 2：提示找不到 Node.js
访问 https://nodejs.org/ 下载并安装

### 问题 3：登录失败
- 确保网络连接正常
- 尝试用 GitHub 登录（最简单）
- 如果邮箱登录失败，用 GitHub 登录

### 问题 4：部署失败
- 查看错误信息
- 重新运行脚本
- 或者联系我帮忙
