## 🚀 快速开始（3 分钟速读版）

### 1. 创建 GitHub 仓库
访问 https://github.com/new → 填写仓库名 → 创建

### 2. 推送代码
```bash
git remote add origin https://github.com/用户名/仓库名.git
git push -u origin main
```

### 3. 在 Vercel 导入
访问 https://vercel.com → 登录 → 导入仓库 → 部署

### 4. 完成！
等待 1-2 分钟，网站上线！

## 💡 重要提示

### 部署前确认：

✅ 已安装 Git
```bash
git --version
```

✅ 已安装 Node.js
```bash
node --version
```

✅ 已安装 pnpm
```bash
pnpm --version
```

✅ 代码已提交
```bash
git status
```

### 部署前准备：

1. **GitHub 账号**
   - 如果没有，去 https://github.com 注册（免费）

2. **记住你的 GitHub 用户名**
   - 后面会用到

3. **决定仓库名**
   - 建议用英文，如：`happy-learning-platform`
